# The varnish package

This package does not contain any useful R code. It serves to hold CSS,
JS, and HTML styling for Carpentries lessons.

## See also

Useful links:

- <https://carpentries.github.io/varnish/>

- Report bugs at <https://github.com/carpentries/varnish/issues/>
